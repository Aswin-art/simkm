# 📊 Sistem Informasi Pemantauan UMKM

Sistem Informasi Pemantauan UMKM adalah aplikasi berbasis web yang dirancang untuk membantu pemerintah atau lembaga terkait dalam **memantau perkembangan, aktivitas, dan kebutuhan UMKM secara real-time**. Aplikasi ini juga memungkinkan pelaku UMKM untuk mengelola data usahanya secara mandiri dan terintegrasi.

## ✨ Fitur Utama

### 👤 Admin
- Ringkasan statistik UMKM (pertumbuhan, wilayah, aktivitas)
- Manajemen data UMKM (verifikasi, legalitas, aktivitas)
- Manajemen program dan bantuan (pendaftaran, penerima, riwayat)
- Notifikasi dan komunikasi (pengumuman, sistem, pesan langsung)
- Pengelolaan akun admin dan hak akses

### 🧑‍💼 UMKM
- Kelola profil usaha dan produk
- Monitoring omzet dan aktivitas usaha
- Mengakses program bantuan dan pelatihan
- Terima notifikasi dari sistem atau admin

## 🛠️ Teknologi yang Digunakan
- **Next.js** — Framework React modern
- **Tailwind CSS** — Styling berbasis utility-first
- **Lucide Icons** — Ikon modern dan ringan
- **next-themes** — Manajemen dark/light mode
- **ShadCN UI** — Komponen UI modular dan responsif
